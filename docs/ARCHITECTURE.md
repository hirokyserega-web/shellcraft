# Архитектура ShellCraft v3

ShellCraft v3 — это распределенная система автоматического крафта для Minecraft (ComputerCraft: Tweaked).

## 1. Компоненты системы
- **Core (Сервер):** Центральный узел. Управляет хранилищем, базой рецептов, очередью задач и диспетчеризацией.
- **Worker (Черепаха):** Исполнительный модуль. Получает задачи, забирает ресурсы, крафтит и сдает результат.
- **Storage:** Система управления периферией (сундуки, бочки, мэ-интерфейсы).

## 2. Транспорт и Буферы
По умолчанию используется **режим буферных сундуков**:
- У каждой черепахи есть входной и выходной сундуки (Input/Output).
- Сервер кладет ресурсы в Input.
- Черепаха забирает ресурсы из Input, крафтит, и кладет результат в Output.
- Сервер забирает результат из Output.

Это исключает необходимость в проводном соединении черепах с общей сетью инвентарей.

## 3. Модель предметов
Используется расширенная модель: `name`, `tags`, `nbt`. Поиск ведется по хешу или через матчер.
